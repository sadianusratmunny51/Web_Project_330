A **Smart Waste Management System** that enables 
- citizens to report waste or recycling requests, 
- allows workers to manage and complete tasks, and
- enables admins to monitor operations - featuring JWT-based authentication, reward points, feedback and rating system, notifications, and a worker leaderboard, built using HTML, CSS, Node.js, Express, and MySQL.

**Tech Stack**
Frontend - HTML, CSS, Vanilla JavaScript
Backend - Node.js, Express.js, 
        - MySQL, JWT Authentication,
        - Multer (image upload)
        - Nodemailer (email & OTP)
        - bcrypt (password hashing)
  
**Installation Guide**
git clone https://github.com/sadianusratmunny51/Web_Project_330.git
cd backend
npm install
npm install --save-dev nodemon

**Run Backend Server**
npm run dev
